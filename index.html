GPA Calculator

This Python project serves as a GPA calculator tailored for Science students. It efficiently computes the Grade Point Average (GPA) by processing exam scores from key subjects such as Mathematics, Biology, Chemistry, Physics, and English. The program extracts student information, including their names and respective scores, from the "students.csv" file. Upon execution, the program takes "students.csv" and the designated student's name as command-line arguments, enabling seamless GPA calculation for individual students.
